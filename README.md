### how can you install it just clone it back to your direct by 
``` bash
  git clone https://github.com/khanhlearnin12/find_files.git
```
- install it by using dpkg such as
```bash
  sudo dpkg -i ./findfiles-builder.deb
```
- to remove
```bash
  sudo dpkg -r ./findfiles-builder.deb
```
### How to use it 
- if not sure what directory or u in just use "~" instead: 
```bash
  find <your_directory> <your_file_name>
```
